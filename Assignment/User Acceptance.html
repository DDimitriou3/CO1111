User Acceptance


1. Visibility of system status :


  • Players are informed when the app wants to use specific features such as location and camera.

  • When players answer a question they are notified if it was correct or wrong

2. The real world and the system match :


  • Questions are fairly simple and straightforward.

  • Some questions are provided with a link for help without forcing the player to search online for meanings.

3. User control and freedom :


  • If the player has made a mistake in answering, it allows them to answer the same question again.

  • Each page is provided with a home button to help players return if they accidentally pressed a button.

4. Consistency and standards :


  • The app is consistent as the layout and design doesnt change when going to the next question.

5. Error prevention :


  • If there is an error it notifies the player that they made a mistake thus making them fix it.

  • If the error is not big, the app itself takes care of it without causing any issues.

6. Recognition rather than recall :


  • QR scans are still shown even if you continue to the next question, it doesnt make the user recall past scans.

7. Flexibility and efficiency :


  • The standard keyboard shortcuts are functional when answering a question (e.g. copying, pasting etc.)

  • The app is available to anyone as the game is pretty straightforward.

8. Aesthetic and minimalistic design :


  • The App User interface contains everything that is needed for the player to play, without unnecessary buttons.

  • Index page contains a menu that allows players to navigate to important parts of the app.

  • The aesthetic of the game is calm on the player, making him enjoy the look of the app.

9. Recognition, diagnostics and recovery from errors :


  • If there is an error, it alerts the player with a translation from programming to regular language so the user can understand.

  • Errors that usually appear also guide players into understanding their mistakes.

10. Help and Documentation :


  • The app is smooth and easy to use.

  • If the player needs any help, there is a help section.
